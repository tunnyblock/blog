负责内容

  • 基于苏宁PCP平台开发Swarm Service创建的逻辑
  • 业务Dockerfile编写，实现Docker Registry的API操作
  • 设计基于Activiti的服务编排引擎，实现根据模板进行任务分发，动态调整执行顺序

使用技术

  • Spring/Spring MVC
  • Mybatis
  • Redis
  • MySQL
  • Swarm
  • Activiti

DCloud 2016.08-2017.02 架构师

项目描述

IBM CDL CaaS容器平台，底层引擎切使用Kubernetes,存储切换到Ceph。主要提供以下内容：

  • 无状态的容器服务，有状态的容器服务
  • 存储卷管理
  • 日志统一处理
  • 基础的CI/CD功能
  • 镜像管理

负责内容

  • 基于Kubernetes的CaaS集群搭建
  • Fabric8io Kubernetes SDK实现与Kubernetes的交互
  • 沿用第一期的Habor Registry,基于secret实现Kubernetes Worker Node镜像的拉取
  • 基于docker stats api与cAdvisor组件实现Kubernetes集群的监控，通过参数动态调整定时任务的执行周期
  • 基于shellinabox实现web端操作容器的Terminal
  • 基于Ceph的存储集群搭建，基于go-ceph封装Ceph RBD的Restful API
  • 基于Ceph存储，实现Kubernetes Persistent Volume的对接，PVC的分配功能
  • 基于ELK的日志统一收集
  • 基于Jenkins的CI/CD平台搭建

使用技术

  • Spring/Spring MVC
  • JDBCTemplate/Mybatis
  • Redis
  • RabbitMQ
  • MySQL
  • Kubernetes
  • Ceph

XCloud 2015.07 - 2016.01 架构师

项目描述

  • IBM CDL基于Openstack的私有云管理，包括如下功能:
  • 虚拟机管理，包括创建，开/关机，重启，远程控制，修改配置，删除
  • 镜像管理，包括公有镜像管理，私有快照管理
  • Floating IP管理
  • 群组划分，和组内资源共享

  • 利用Openstack4j组件实现Openstack各个对象的操作
  • 使用Quartz调度任务，实现虚拟机状态的自动管理

使用技术

  • Spring/Spring MVC
  • JDBCTemplate/Mybatis
  • Redis
  • RabbitMQ
  • MySQL
  • Openstack/Openstack4j

Mobile Cloud 2013.12 - 2014.06 架构师

项目描述

IBM CDL 手机云测试平台，类似于Testin的平台，提供Mobile测试云，包括如下功能:

  • Android真机的申请和远程控制
  • iPhone真机的申请和远程控制
  • Linux虚拟机的创建和Android真机的挂载
  • Mac虚拟机的创建和iPhone真机的挂载

负责内容

  • 利用VMware USB Driver接口，动态实现手机与虚拟机挂载和断开
  • 利用noVNC，google adb和libmobiledevice实现Android，iPhone，iPad的Web远程操作
  • 基于RabbitMQ解耦前台申请和后台资源的操控，包括手机与虚拟机的挂载，断开等操作

使用技术

  • Spring/Spring MVC
  • JDBCTemplate/Mybatis
  • RabbitMQ
  • DB2
  • VMWare Perl SDK
